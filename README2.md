# Ejercicio Funciones

## Descripción
Este programa calcula el promedio de tres notas utilizando una función con parámetros y retorno de valores.

## Lenguaje
Python

## Cómo funciona
- Se ingresan tres notas
- Se llama a una función
- La función calcula el promedio
- Se muestra el resultado
